Agora vamos falar de formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b>(Não semântico)
Nesta frase, tenho um termo em Destaque usando a tag <strong>(semântico, a Significado)

Itálico/Ênfase

Nesta frase, temos um termo em Itálico usando a tag <i> (Não semântico)
Nesta frase, temos um termo em Ênfase usando a tag <em>(semântico, a Significado)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um pequeno neste parágrafo, usando a teg <big>(uma tag defasada) e a tag <small>(uma tag ainda usada)

Texto deletado

Podemos marcar um texto como excluido usando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois (semântico, a Significado)
Existe também o sublinhado com a tag <u>(não semântico)

Texto sobrescrito

Para inseri coisas do tipo x2+3, usando a tag <sup>

Texto subscrito

Para inseri coisas do tipo H2O, usando a tag <sub>

Código-fonte e Pré-formatado

O comando document.getElementById('teste') é escrito em líguagem JavaScript, usamos a tag <code> para formatação da leitura do código(Mas para obter um formato identalizado você usa a teg <pre> e depois usa <code> para uma melhor leitura)
